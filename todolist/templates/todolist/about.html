About

The personal planner is a web application built using Django. It provides the user with an interactive personal assistant experience, and displays the user inputs in a clear and minimal layout on the dashboard page.

Planner items are organized by category and sorted by due date. To delete an item, clicking on the item title will bring up a Details page with a Delete button.

This planner is a personal project built with goal of developing my skills in web development and Django. The source code can be found on GitHub. If you would like to see my other work or contact me, visit my personal website.
